More fun with Panos

Mon, Jun 14, 2004

Since writing the article in the Luminous Landscape, I've run across Max Lyon's forums covering all things pano.  Cool stuff I've discovered or been alerted to:

  • A really sweet cheap DIY pano tripod head using 8020 aluminum extrusion stock.  I built one of these last week and it works great!
  • Autopano -- a helper to automatically find the control points for panotools.  PTAssembler can be set up to call this directly.
  • Enblend -- a tool to automatically do the blending between images based on the spatial frequency of the image at the seam.
  • Hugin -- a free (as in speech) appliaction for driving PanoTools.  There is also an experimental replacement for PanoTools called nona.  I haven't played with this yet but it is interesting to see this stuff develop.  If this weren't GPL'd and similar to my real job, I'd love to get involved and write some code.
  • nGallery -- I installed this on an ASP.Net server and I'll be posting some new pictures soon.  It was super easy to get up and running and looks configurable.  This is something that I could get involved in and write some code.

Hopefully I'll have some example panos put together using the new setup soon.
